# Odin Calculator

A simple calculator web application built as part of The Odin Project curriculum.

## Description

This project implements a basic calculator with addition, subtraction, multiplication, and division operations. It uses HTML, CSS, and JavaScript to create an interactive web-based calculator.

## Features

- Basic arithmetic operations: addition, subtraction, multiplication, and division
- User-friendly interface
- Responsive design

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)

## How to Run

1. Clone the repository or download the files.
2. Open `index.html` in your preferred web browser.
3. Use the calculator interface to perform calculations.

## Project Status

This is a work-in-progress project. The core functionality is being implemented step by step.
